📖 คู่มือติดตั้งระบบแจ้ง IP กลับส่วนกลาง
(รองรับ Windows 10 และ 11)

⚙️ ส่วนที่ 1: การสร้างไฟล์ตั้งค่า

1. สร้างไฟล์สคริปต์ที่หน้า Desktop
คลิ๊กขวาที่พื้นที่ว่างบน Desktop เลือก New > Text Document
2. นำโค้ดไปวางในไฟล์
เปิดไฟล์ขึ้นมา กดปุ่ม Copy ด้านล่าง แล้วนำโค้ดไปวาง

3. บันทึกไฟล์ (Save)
- ไปที่เมนู File > Save As...
- ช่อง File name พิมพ์ว่า update_ip.ps1
- ช่อง Save as type เลือกเป็น All Files (*.*)
- กดปุ่ม Save จะได้ไอคอนสีฟ้าๆ ไว้ที่หน้าจอ
✅ 4. วิธีทดสอบการทำงานของสคริปต์ (แนะนำให้ทำก่อนไปส่วนที่ 2)
เพื่อให้ชัวร์ว่าสคริปต์ไม่พัง และแก้ชื่อสาขาถูก ให้ทดสอบดังนี้:
1. เปิด Windows PowerShell (กดปุ่ม Start แล้วพิมพ์ค้นหา powershell)
2. พิมพ์ตัว & แล้วเว้นวรรค 1 ที จากนั้นเอาเมาส์ลากไฟล์ update_ip.ps1 จากหน้าจอ มาปล่อยลงในหน้าต่าง PowerShell แล้วกด Enter 1 ครั้ง
3. หากทำงานเสร็จโดย ไม่มีข้อความ Error สีแดง ขึ้นมา ถือว่าโค้ดสมบูรณ์แล้ว! (สามารถแจ้งแอดมินให้เช็กในระบบได้เลย)

🚀 ส่วนที่ 2: ตั้งค่าให้รันอัตโนมัติ (Task Scheduler)

ขั้นตอนนี้จะช่วยให้คอมพิวเตอร์รันสคริปต์อัปเดต IP ให้เองทุกๆ 10 นาที โดยไม่โชว์หน้าต่างรบกวนการทำงานครับ

1. เปิด Task Scheduler
กดปุ่ม Start (รูป Windows) พิมพ์ค้นหาคำว่า Task Scheduler แล้วเปิดขึ้นมา
2. สร้างงานใหม่ (Create Task)
ที่เมนูด้านขวามือ ให้คลิ๊กที่ Create Task...
3. ตั้งค่าหน้า General (หน้าแรก)
- Name: พิมพ์ว่า Auto Update IP
- Security options: เลือก Run whether user is logged on or not
- ติ๊กถูกช่อง Do not store password... (อยู่ด้านล่างสุด) เพื่อซ่อนหน้าต่างดำตอนรัน
- ติ๊กถูกที่ Run with highest privileges
- ติ๊กถูกที่ Hidden (มุมซ้ายล่างสุด)
4. ตั้งค่าเวลาหน้า Triggers (ตั้งเวลาทำงาน)
- กดแท็บ Triggers ด้านบน > กดปุ่ม New...
- Begin the task: เลือก On a schedule (ตั้งเวลา) หรือ At startup
- กรณีเลือก On a schedule ให้ตั้งค่าด้านซ้ายเป็น Daily (ทุกวัน)
- สำคัญที่สุด: ติ๊กถูกช่อง Repeat task every: แล้วพิมพ์ 10 minutes
- for a duration of: เลือก Indefinitely
- กด OK
5. ใส่คำสั่งในหน้า Actions (ตั้งคำสั่ง)
- กดแท็บ Actions ด้านบน > กดปุ่ม New...
- Program/script: พิมพ์ powershell.exe
- Add arguments: ให้ก๊อปปี้ข้อความด้านล่างนี้ไปวาง:
-ExecutionPolicy Bypass -WindowStyle Hidden -File "C:\Users\Admin\Desktop\update_ip.ps1"
*อย่าลืมเปลี่ยน C:\Users\Admin\Desktop\update_ip.ps1 เป็นที่อยู่ไฟล์จริงๆ ของเครื่องคุณ
- กด OK
6. บันทึกและเริ่มใช้งาน
กด OK หน้าต่างสุดท้าย เป็นอันเสร็จสมบูรณ์! ระบบจะแอบเช็ค IP ให้ทุก 10 นาทีครับ

🛑 ส่วนที่ 3: วิธีการปิดการทำงาน หรือลบสคริปต์ทิ้ง

1. เปิด Task Scheduler แล้วคลิ๊กเมนู Task Scheduler Library (ฝั่งซ้ายมือ)
2. หาชื่อ Auto Update IP
หาชื่อที่เราตั้งไว้ในรายชื่อตรงกลางจอ แล้ว คลิ๊กขวา
3. เลือกคำสั่งที่ต้องการ:
- เลือก Disable = ปิดการทำงานชั่วคราว (สคริปต์หยุดทำงาน เผื่อวันหลังอยากเปิดใหม่)
- เลือก Delete = ลบทิ้งถาวร
